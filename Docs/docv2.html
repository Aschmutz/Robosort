RenderInfoBundle Schema Documentation

RenderInformation ($id: RenderInfoBundle.json)

Description: RenderInformation for the render Queue

Property Description
RenderInformation (Required) Reference to the nested RenderInformation schema definition.

RenderInformation ($id: RenderInformation)

Description: The Information for a RenderQueue

Property Description
inputFolder The local path to the Folder Containing the Alembic Models (Required).
outputFolder The local path to the Folder where the rendered images are placed (Required).
operations The Render-operations that are to be done with the files (Required).
operations[] Array of RenderOperation objects.

RenderOperation ($id: RenderOperation)

Description: A Render operation consisting of the Render details and the amount of pictures to render

Property Description
simulationTime (Number) The Time that the physics Simulation will run for (Required).
amount (Integer) The amount of images that are to be rendered with these settings (Required, minimum: 0).
rotationType (Enum) The type of Rotation that is needed. The Options are "random", "initial", "custom", "computed" (Required).
simulationSpeed (Number) The speed at which the simulation is run (default: 1, minimum: 0).
rotation (Object) The rotation of the object.
